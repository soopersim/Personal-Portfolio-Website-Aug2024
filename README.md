# Personal-Portfolio-Website-2024
# Overview
This is the code for my personal portfolio webpage. It showcases a little introduction, notable projects, and contact information, providing a clean and responsive design to highlight their skills and achievements.

# Features
- Hero Section: Introduction with a profile image.
- Projects Showcase: A section displaying key projects with descriptions and links.
- Footer: Includes links to terms of service, privacy policy, and copyright information.

# Technologies Used
- HTML5
- CSS3
- Remix Icon
- Google Fonts
